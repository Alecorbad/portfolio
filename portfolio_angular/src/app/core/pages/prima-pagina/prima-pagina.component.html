Prima settimana di PCTO

  • La prima settimana di stage mi ha permesso di imparare meglio il linguaggio HTML e CSS , grazie, anche, ai manuali forniti dall'azienda,

  • Mi ha permesso di creare pagine HTML e CSS.

  • Durante il primo giorno di stage mi è stato affidato il compito di installare GitFork dell'IDE GitHub , Visual Studio Code e Node.js all'interno del computer aziendale,

    con implemento di documentazione HTML, fornita dall'azienda.

  • Formattazione fogli HTML e CSS su Visual Studio Code.

  • Per la realizzazione dei file HTML sono state usate le classi, definite in ogni file CSS.

    Per integrare il codice CSS all'interno del codice HTML, è stato usato il tag link href all'interno del file HTML.

  • Per integrare il font è stato utilizzato il tag font face direttamente all'interno del file HTML.

  • Per porre al centro il titolo, è stata usata la classe title-container>div.

  • All'interno della classe title-container>div> h1 , utilizzata per determinare la grandezza del titolo, il tag color identifica il colore del titolo.

    Per creare le ombre intorno al titolo è stato usato usato il tag text-shadow, all'interno della classe title-container>div h1.

  • Per creare la sovrapposizione delle due immagini, nella home page, sono state usate due classi, shadow e image.

    All'interno della classe shadow, è stato inserito il tag z-index, che dimostra il livello dell'immagine, z-index: -1 significa che l'immagine sarà posta sotto un'altra immagine, e la box-shadow

    per determinarne l'ombra.

    All'interno della classe shadow è stato utilizzato il tag filter: brightness, ovvero la luminosità dell'immagine sottostante.

    All'interno della classe image è stato inserito il tag z-index: 0, in modo che l'immagine sia sovrapposta rispetto alla seconda immagine di sfondo.

    Le immagini sono state allegate attraverso il tag src, direttamente all'interno del file HTML.

  • Per cambiare il cursore del mouse quando si clicca sul link, è stata utilizzata la classe cursor, con all'interno il tag cursor:pointer.

  • Per cambiare il colore dei link, è stato utilizzato il tag font color, all'interno del file HTML.

  • Per la creazione dei bottoni è stata usata la classe buttons, con, al suo interno, display: flex, che permette di migliorare il posizionamento dei vari bottoni.

    Per la grandezza dei bottoni sono stati usati i tag height, ovvero l'altezza, e width, ovvero la larghezza, con misure in rem. 1 Rem corrisponde a 16 pixel.

    All'interno della classe buttons, è stato inserito il primo colore di background.

    Per creare l'effetto sfumato all'interno dei bottoni è stata usata la classe hover , con, all'interno, il secondo colore di background.

  • Per permettere la funzionalità dei bottoni, è stato usato il tag a href, direttamente all'interno del file HTML.
